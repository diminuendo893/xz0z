## 前言

徐州市第零中学，旨在通过互联网，面向公众传播教育。

学校定位为高级中学，但并不局限于高中，任何有价值，值得展开教学的内容都可以成为一门课程。

目前学校正在建设中，如果你愿意为本校的建设作贡献，你可以与我们协作并提交分支，通过审核后校方或将合并PR。你也可以创建你自己的仓库，再把网址告诉我们，通过审核后校方将使用此链接。详情请见[]()。
